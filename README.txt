## Variables necesarias para tu grupo de chat

Para que el bot funcione correctamente en tu grupo de chat, asegúrate de configurar las siguientes variables en tus secretos:

- **`LINK`**: La URL del Moodle. Asegúrate de que los servicios estén activos.
- **`TOKEN`**: El token del estudiante en Moodle.
- **`USER_ID`**: El ID del usuario del estudiante en Moodle.
- **`TELEGRAMTOKEN`**: El token proporcionado por el BotFather en Telegram.
- **`CHAT_ID`**: El ID del grupo de chat en Telegram donde se notificarán las tareas.

Asegúrate de reemplazar cada valor con la información correspondiente antes de iniciar el bot.
